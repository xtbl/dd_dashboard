dibo
============

frontend didi dashboard



Requisitos:
---------------------

- [node.js]
- [npm]
- [Bower]
- [Compass]
- [Grunt]

Instalación:
---------------------
```
$ npm install
$ bower install
```
Levantar server y aplicación
```
$ cd app
$ grunt server
```
Build
```
$ grunt build
```
